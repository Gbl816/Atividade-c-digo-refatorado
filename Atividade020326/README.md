# Projeto Banca Central

Este repositório contém uma página HTML simples chamada `index.html` (e uma versão refatorada `index_refatorado.html`) que representa a página inicial de uma banca de revistas fictícia. Há também uma folha de estilos CSS (`style.css`) responsável pela aparência da página.

## Estrutura de arquivos

- `index.html` - código HTML original da página principal.
- `index_refatorado.html` - versão com marcação semântica ou reformatada (se existir).
- `style.css` - estilos utilizados pela página HTML.
- `revista.jpg` - (opcional) imagem de exemplo usada no conteúdo.
  
	**Nota:** há também imagens no diretório (por exemplo arquivos [lighthouseprint.png](./lighthouseprint.png) e [autentificadorprint.png](./autentificadorprint.png)) que representam os resultados do projeto; essas imagens são legendadas como **"Resultados do projeto"**.

## Como visualizar

1. Abra o arquivo `index.html` em um navegador web (Chrome, Firefox, Edge, etc.).
2. Certifique‑se de que `style.css` está no mesmo diretório para que o layout seja aplicado.
3. Se desejar testar a outra versão, carregue `index_refatorado.html` em vez de `index.html`.

## Funcionalidades básicas

- Cabeçalho com logo e navegação simples.
- Conteúdo central com título, parágrafos, imagem e subtítulo de destaque.
- Rodapé com endereço e telefone.
- Estilos CSS que definem fonte, cores, espaçamento e responsividade básica.

## Customização

- Edite `index.html` para modificar textos, links ou estrutura.
- Ajuste `style.css` para alterar cores, tamanhos, margens ou adicionar novos estilos.
- Adicione outras páginas (por exemplo, `produtos.html`, `contato.html`) e links no menu.

## Acessibilidade

- A imagem principal (`revista.jpg`) inclui um atributo `alt` com descrição.
- As imagens em `img/` (`.png` ou outras) também devem ter `alt` e são usadas como resultados do projeto.
- É recomendável continuar aplicando boas práticas de HTML semântico e atributos ARIA se necessário.

Este projeto serve como exemplo básico para estudos ou como ponto de partida para um site estático simples.
